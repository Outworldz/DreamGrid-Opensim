
This folder is served to the web at the path http://yoururls:8002/bin/data  by Divi's Wifi program via robust.

The following file types are supported:

.jpg image/jpg
.gif image/gif
.png image/png
.ico image/ico
.css text/css
.txt text/plain
.xml text/xml
.js application/javascript
.exe application/exe

All others are MIME type text/html

The files here are used for Statistics and Teleport signs, and avataranimations.xml is needed for the system to boot.
